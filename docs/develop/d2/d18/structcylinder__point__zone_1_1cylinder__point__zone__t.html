Detailed Description

A cylindrical point zone.

As defined here, a cylinder is described by its two end points and its radius, specified in the json file as e.g. "start": [<x0>, <y0>, <z0>]", "start": [<x1>, <y1>, <z1>]", "radius": <r>.
