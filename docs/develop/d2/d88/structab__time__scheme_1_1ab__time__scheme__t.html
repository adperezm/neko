Detailed Description

Explicit Adam-Bashforth scheme for time integration.

For a contant time-step this corresponds to the following schemes for order 1 to 3:

  • Order 1: \( u^{n+1} = u^n \)
  • Order 2: \( u^{n+1} = 2u^n - u^{n-1} \), linear extrapolation
